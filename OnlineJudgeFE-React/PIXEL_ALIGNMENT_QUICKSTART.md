## 🔧 修复流程

### 每个差异的处理

```bash
# 1. 备份（必须！）
cp src/pages/XXX.tsx src/pages/XXX.tsx.backup_$(date +%Y%m%d_%H%M%S)

# 2. 修改（使用 search_replace）
# 根据工具提供的"修复建议"进行修改

# 3. 提交（立即）
git add src/pages/XXX.tsx
git commit -m "fix: [属性名]设为[8080值] (8080精确值)"

# 4. 验证（刷新工具页面）
# 确认差异数减少

# 5. 重复直到差异数 = 0
```
