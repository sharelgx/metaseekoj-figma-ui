# 🔍 Scratch 编辑器登录入口位置说明

## 📍 **登录按钮位置**

访问：`http://localhost:8081/classroom/scratch/editor`

### **在 Scratch 编辑器中找到登录按钮**

```
┌─────────────────────────────────────────────────────┐
│ 🎨 Scratch 编辑器                   [登录] ← 这里！ │ 绿色菜单栏
└─────────────────────────────────────────────────────┘
```

**具体位置**：
- **顶部绿色菜单栏**（Scratch 编辑器最上方）
- **最右侧**
- 显示文字：**"登录"** 或 **"Sign in"**

---

## 🎯 **登录步骤（详细）**

### **步骤 1：定位登录按钮**

打开 `http://localhost:8081/classroom/scratch/editor`

**观察页面**：
- 最顶部应该有一条**绿色的菜单栏**
- 菜单栏左侧有：文件、编辑等菜单
- 菜单栏**右侧**有：登录按钮

**如果看到**：
```
[加入 Scratch] [登录]
```
说明您**未登录**，可以点击登录

**如果看到**：
```
[👤头像] 用户名 ▼
```
说明您**已登录**

---

### **步骤 2：点击登录**

点击**"登录"**按钮后：
- 🎯 自动跳转到登录页面
- 🎨 看到精美的登录界面
- 📝 有用户名和密码输入框

---

### **步骤 3：输入账号信息**

**如果还没有账号**：
1. 点击"立即注册"
2. 填写注册信息
3. 注册成功后回到登录页

**如果已有账号**：
1. 输入用户名
2. 输入密码
3. 点击"登录"按钮

---

### **步骤 4：登录成功**

登录成功后：
- ✅ 自动返回 Scratch 编辑器
- ✅ 右上角显示您的头像和用户名
- ✅ 现在可以保存项目了

---

## ❓ **常见问题**

### **Q1：看不到登录按钮**

**可能原因**：
- Scratch 编辑器还在编译中
- 页面缓存

**解决方案**：
1. 刷新页面（Ctrl + F5 强制刷新）
2. 或直接访问登录页：
   ```
   http://localhost:8081/login?return_url=/classroom/scratch/editor
   ```

---

### **Q2：点击登录没反应**

**解决方案**：
1. 打开浏览器控制台（F12）
2. 查看是否有错误
3. 或直接访问：`http://localhost:8081/login`

---

### **Q3：想直接跳转到登录页**

**直接访问**：
```
http://localhost:8081/login
```

登录成功后会自动返回编辑器

---

## 🚀 **快速登录（最简单）**

如果您不想在编辑器中找登录按钮：

### **直接访问登录页**：
```
http://localhost:8081/login?return_url=/classroom/scratch/editor
```

**步骤**：
1. 输入用户名
2. 输入密码
3. 点击"登录"
4. ✅ 自动返回编辑器

---

## 💡 **提示**

如果您是**第一次使用**，需要先注册账号：

**访问注册页**：
```
http://localhost:8081/register
```

**或者在登录页点击**：
```
"立即注册"按钮
```

---

**总结**：在编辑器页面登录最简单的方法是点击 Scratch 编辑器右上角的"登录"按钮，它会自动跳转到登录页面！🎯
